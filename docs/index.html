JPaint
Where the only limit to what you can create is your imagination.
JPaint has a variety of useful tools to work with.

JPaint is a paint program that you can use to create anything your imagination desires! It comes with a variety of tools, such as a pencil, a paintbrush, a shape tool, and more!

JPaint was written using Java's Swing GUI widget toolkit and uses a model-view-controller architecture. It opens and saves 32-bit PNG files.

JPaint can be run on a Mac or PC with Java 7 or later installed. You can download the program as an executable .jar file, or view the source code on GitHub.
